## 🚀 Getting Started (3 Steps)

### Step 1: Quick Start (5 min)
```bash
# Install dependencies if needed
pip install -r requirements.txt

# Run the app
streamlit run src/streaming/app.py

# Open browser to http://localhost:8501
```

### Step 2: Test Detection (5 min)
1. Check sidebar: Model shows "● Ready"
2. Upload image from AMR_Dataset/Test_image/
3. Click "🚀 Run Detection"
4. View results and logs

### Step 3: Review Documentation
- [QUICK_REFERENCE.md](QUICK_REFERENCE.md) for daily use
- [TESTING_GUIDE.md](TESTING_GUIDE.md) for troubleshooting
